## Development

### Local Development Environment Setup and Installation

*  Clone repository:
```bash
git clone https://github.com/Rise-Vision/chrome-os-player-poc.git
```

*  Install:
```bash
npm install
```

* Build:
```bash
npm run watch
```

* Load the packaged app on `chrome://extensions`
